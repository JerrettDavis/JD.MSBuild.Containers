# Issue Title  
NuGet Pack Fails with NU5017 - Package has no dependencies nor content  

# Issue Body  
The CI build is failing during the pack step with the following error:

```
error NU5017: Cannot create a package that has no dependencies nor content.
```

**Build Details:**  
- **Workflow:** CI (.github/workflows/ci.yml)  
- **Job URL:** https://github.com/JerrettDavis/JD.MSBuild.Containers/actions/runs/21052896461/job/60542907373  
- **Commit:** 3b4ea4b19de373f1e180235b4ba376ed069afbc9  
- **Failed Step:** NuGet pack for `JD.MSBuild.Containers.csproj`  

**Root Cause:**  
The project file `src/JD.MSBuild.Containers/JD.MSBuild.Containers.csproj` is not configured to include any content, assemblies, or dependencies in the resulting NuGet package.  

**Solution:**  
Update the `.csproj` file to include:
1. Build output (DLL/assembly) if this is a library package  
2. MSBuild props/targets files if this is an MSBuild SDK package  
3. Content files with appropriate `Pack="true"` and `PackagePath` attributes  

Example for MSBuild SDK packages:
```xml
<ItemGroup>
  <None Include="build\**" Pack="true" PackagePath="build\" />
  <None Include="buildTransitive\**" Pack="true" PackagePath="buildTransitive\" />
</ItemGroup>
```
